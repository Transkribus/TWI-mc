{% trans 'How to start a crowd sourcing project for your collection' %}

Follw the steps below using the Transkribus desktop application to prepare a collection for a crowd sourcing project.

Check list

  • A Transkribus account. Get an account.
  • The Transkribus Desktop appliction. Download.
  • A Collection within Transkribus. Read more.

Preparation phase in Transkribus desktop

Within Transkribus an owner has the ability to open up a collection for crowd-sourced transcription. This can only be done once the baselines have been detected and recorded. Crowd sourcing can be started without any automatic transcription (ie with a blank transcript) or after HTR has been run (for the purposes of improving transcription quality). The whole collection will be made available so unless the owner wishes all the documents to be made available they should consider creating a new collection that will include only those documents they wish to be made avialable for public transcription.

Add manuscripts

Upload or select from existing collections the select the manuscripts that you wish to make available to the crowd-sourcing platform

Line segmentation

It is essential that line segmentation is performed on the documents in order to give the prepare the transcripts for the addition of text. This is done as a batch process after upload.

Polygon2Baseline tool

This tool will create snippets of the doumnet image that correspond to the lines detected in the previous step.

Mark as crowdsourcing

At this point we are ready to mark the collection as available for crowd-sourcing. This can be done within the "Edit Collection Metadata" window by owner or editor. Unselecting this option will mean that crowd-sourcing is closed again and transcription can only be done via the expert tool. The owner can write a description of the collection that will be displayed within the crowd-sourcing platform to explain what is the collection and why it may be of interest to public transcribers.

Choose a representative image

Owner can choose and upload an image to represent the collection on the crowd-sourcing platform. This image will be displayed alongside the descrption and as a thumbnail in the browsing interface.

Run HTR

Depending on the nature of your crowd sourcing project you may wish to seed the transcripts with the output tfrom the Transkribus Handwritten Text Recognition (HTR) process. In such cases Transkribus will process the images and populate the transcripts so that your "crowd" will then have the task of correcting any errors that have arisen. For more on how to get the best out of the HTR process see...

Define a tag set

The crowd-sourcing interface allows transcripts to be enhanced by tagging words and phrases with metadata. This is a task particularly suited to public participation. Transkribus has a set of predefined core tags, but a collection owner can spefic their own tags that may be suitable for the given collection.
